Understanding CSS Fonts

What Are CSS Fonts?

CSS fonts allow you to style text on a webpage. By using different font properties, you can control the appearance of your text, including the font type, size, weight, and more. Let's explore the key font properties in CSS:

  • font-family - Specifies the typeface used for the text.
  • font-size - Defines the size of the text.
  • font-weight - Sets the thickness of the font.
  • font-style - Defines the style of the font (e.g., normal, italic).
  • line-height - Controls the space between lines of text.
  • text-align - Aligns the text within its container.

Font Family

The font-family property specifies the typeface for the text. You can list multiple font families as fallbacks, in case the preferred one is unavailable.

This text is styled with the "Georgia" font. 
p {
    font-family: 'Georgia', serif;
}

Font Size

The font-size property sets the size of the text. It can be specified in various units such as pixels (px), ems (em), or percentages (%).

This text has a font size of 24px. 
p {
    font-size: 24px;
}

Font Weight

The font-weight property controls the thickness of the text. Common values are normal (400) and bold (700), but you can specify any value from 100 to 900.

This text is bold (font-weight: bold). 
p {
    font-weight: bold;
}

Font Style

The font-style property is used to apply a normal, italic, or oblique style to the text.

This text is italic (font-style: italic). 
p {
    font-style: italic;
}

Line Height

The line-height property controls the space between lines of text. It can be set using unitless numbers, percentages, or any length unit.

This text has a line height of 1.8.
This creates more space between lines of text. 
p {
    line-height: 1.8;
}

Text Align

The text-align property sets the alignment of text within its container. Common values are left, right, center, and justify.

This text is centered (text-align: center). 
p {
    text-align: center;
}
